# ChoLib Documentation

[ChoLib](https://github.com/tantaihaha4487/ChoLib) is a lightweight Fabric mod library that lets players trigger abilities by rapidly pressing shift while holding registered items.

## Documentation

Full documentation is available at [https://tantaihaha4487.github.io/ChoLib/](https://tantaihaha4487.github.io/ChoLib/)

## Development

```bash
npm install
npm run docs:dev    # Start dev server
npm run docs:build  # Build for production
npm run docs:preview # Preview production build
```
